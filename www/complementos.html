¿Qué necesitamos para utilizar cordova?

Cordova para Android requiere el SDK de Android que se puede instalar en OS X, Linux o Windows. Consulte los requisitos del sistema del SDK de Android . El último paquete de Android de Cordova admite hasta el nivel de API de Android 28.

Además requerimos de las siguientes herramientas:

Android SDK, Gradle, Java y Node JS.
