Cordova

¿Qué es cordova?

Apache Cordova es un framework de licencia libre que cuenta con muchas Apis de diversos dispositivos móviles para desarrollar aplicaciones nativas dentro de un smartphone. Cada ves está tomando más énfasis en el mundo de los programadores y es que para el desarrollo de las aplicaciones se utilizan las tecnologías web HTML, CSS y JavaScript.
